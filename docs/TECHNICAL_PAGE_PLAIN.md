# Union Rig — Technical Page (Plain)

Union Rig is a mono-in / stereo-out guitar instrument with a fixed signal structure, a 12-sound working set, two scenes per sound, and a performance looper.

## Fixed structure
Dynamics → Drive → Character/Motion → Stereo Width → Space → Output Shaping

## Working set & scenes
- 12 sounds loaded (no banks)
- Each sound has Scene A and Scene B
- Footswitches: Looper / Scene / Next / Previous

## Preference memory
Local Preference Vector: macro-level biases learned only from explicit commits.

## LLM role
Optional assistant proposes bounded deltas (control plane only). Never in audio thread.

## Power
IEC C14 inlet with internal AC–DC PSU.
