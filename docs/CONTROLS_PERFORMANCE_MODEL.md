# Union Rig — Controls & Performance Model (Locked)

## Knobs (macros)
Touch, Heat, Body, Depth, Motion, Tempo

### Cluster intent
- Left cluster (Tone/Feel): Touch / Heat / Body
- Right cluster (Space/Time): Depth / Motion / Tempo

## Footswitches (left→right)
1) Looper
2) Scene
3) Next Sound
4) Previous Sound

### Scene switch behavior
- Tap: Scene A ↔ Scene B
- Hold: momentary morph toward opposite scene; release returns smoothly

### Sound navigation
- Tap Next: advance through 12 sounds (wraparound)
- Tap Previous: go back (wraparound)

## Working set
12 sounds × 2 scenes = 24 playable states with no banks or browsing.
